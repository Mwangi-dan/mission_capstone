How the Model Works

The model is a convolutional neural network (CNN) designed for binary classification, trained specifically on endoscopic images to identify "Cancerous" and "Non-Cancerous" cases. Leveraging deep learning techniques, the model extracts high-level features from images, such as textures and patterns, that are indicative of cancerous lesions.

  • Model Type: Convolutional Neural Network (CNN)
  • Input Dimensions: 128x128 pixels
  • Output Classes: Cancerous, Non-Cancerous
